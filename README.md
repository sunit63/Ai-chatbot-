# 🧠 HOW IT WORKS

### 🤖 Chatbot Flow
```mermaid
graph TD;
A[User Input] --> B{Known?}
B -->|Yes| C[Reply from Memory]
B -->|No| D[Learn New Response]
D --> E[Save to JSON]
```

### 📩 Spam Detection Flow
```mermaid
graph TD;
A[Input Text] --> B[TF-IDF Vectorizer]
B --> C[Naive Bayes Model]
C --> D[Spam / Not Spam]
```
